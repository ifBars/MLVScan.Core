# MLVScan.Core Documentation

`MLVScan.Core` is the environment-agnostic scanning engine that powers the wider MLVScan ecosystem. This site combines the library API reference with the core architecture and environment-separation guidance used across consumers.

- Cross-platform static analysis for .NET assemblies
- Rule-driven detection with deterministic results
- Strict separation between extraction, orchestration, and rule evaluation
- Stable public Core APIs for downstream consumers
